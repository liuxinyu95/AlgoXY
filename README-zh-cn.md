AlgoXY是一本关于初等算法和数据结构的书。这本书不仅仅关注命令式的方法，还包括纯函数式的算法和数据结构。所有的算法都同时使用了数学公式和伪代码来描述。

本书中文版《算法新解》已出版：
http://www.ituring.com.cn/book/1907

本书提供了C、C++、Haskell、Python、和Scheme/Lisp的源代码作为实现的例子供读者参考。为了增加Java和Scala的例子代码，我们增加了一个名为jvm的分支。
